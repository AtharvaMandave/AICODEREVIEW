# Frontend Setup

## Environment Variables

Create a `.env.local` file in the client directory with:

```
NEXT_PUBLIC_API_URL=http://localhost:5000/api
```

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

The app will run on http://localhost:3000
